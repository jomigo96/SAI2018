# Laboratório 1: Route distance

## Descrição

### Instruções de uso

Compilar o programa com 
```
make
```

executar com
```
./prog <path to input file>
```

O ficheiro de input deve ter o formato:
```txt
<Waypoint code>;<Latitude (deg)>;<Longitude (deg)>;<Height (ft)>
...
```
